# The Color Cult
Chris Mok, Hannah Zhang, Chaewon Bae

# Paper Details
We implemented the paper "[Color Me Noisy: Example-based Rendering of Hand-colored Animations with Temporal Noise Control](https://dcgi.fel.cvut.cz/home/sykorad/cmn.html)" by J. Fiser et. al.

# Slides
The slides for our presentation including the demo video can be found [here](https://docs.google.com/presentation/d/1suS7xtgWvyVkrgnMnadNyxkXsWuZsL-S68O9wYB72Z8/edit?usp=sharing)

# Extensions
We implemented local source control for subject/texture boundaries to ensure that the edges of the subject in the animation only sampled patches from the edges of the textures for more realistic texturing.
